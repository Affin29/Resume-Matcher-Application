# Resume-Matcher-Application
Smart Resume Matcher is a Streamlit-based NLP web app that checks how well a resume matches a job description using semantic similarity and machine learning techniques.
It helps job seekers improve resumes and helps recruiters quickly shortlist candidates.

#Features
.Resume vs Job Description matching
.NLP text cleaning (stopword removal, normalization)
.Semantic embeddings using Sentence Transformers
.Cosine similarity score calculation
.Match percentage with feedback level
.Simple and interactive Streamlit interface
.Supports ranking multiple resumes from dataset

 #How It Works
.Paste job description
.Paste resume text
.System cleans the text
.Converts text into semantic vectors
.Computes cosine similarity
.Displays match score and strength level
